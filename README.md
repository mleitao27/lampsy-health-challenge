## Setup
1 - Install git

2 - Install node

3 - Clone repo

4 - Enter my-app directory
```
cd my-app
```
5 - Install dependencies
```
npm i
```
6 - Run app
```
npx expo start
```
7 - Scan QR code displayed on the terminal
